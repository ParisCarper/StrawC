## Common Commands for mcserver


## -- Hot Commands -- 

## Start Server
java -Xmx5120M -Xms3072M -jar paper-1.19.3-396.jar


## Watch Temp
watch -n 1 vcgencmd measure_temp

## Watch Clock Speed
watch -n 1 vcgencmd measure_clock arm

## Remove without Error
rm -rf "name"

## Check storage left on server
df -h
